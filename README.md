# Little Lemon Booking System

A Django-based web application developed as part of the Meta Back-End Developer Professional Certificate. The project provides a restaurant booking system with REST API endpoints for managing reservations and restaurant operations.

## Features

- Restaurant table booking system
- Django REST Framework APIs
- SQLite database integration
- Modular Django application structure
- Static file support
- Easy to extend and customize

## Technologies Used

- Python 3
- Django
- Django REST Framework
- SQLite
- Pipenv
